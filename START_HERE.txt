💡 نصيحة سريعة / Quick Tip:
================================================================================

لتوليد عرض PowerPoint من ملف PDF:
To generate a PowerPoint from a PDF file:

   python launcher.py yourfile.pdf --format powerpoint --title "My Presentation"

لتوليد فيديو بالعربية:
To generate an Arabic video:

   python launcher.py yourfile.pdf --format video --title "فيديو توعوي" --language ar

لتوليد جميع الصيغ (فيديو + PowerPoint + Word + Excel + PDF):
To generate all formats (Video + PowerPoint + Word + Excel + PDF):

   python launcher.py yourfile.pdf --format all --title "Complete Package"
